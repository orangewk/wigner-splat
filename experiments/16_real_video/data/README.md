# carousel_frames.npz — 実動画データの来歴

- 出所: orange 撮影の手持ち歩行動画(2026-07-16 提供、iPhone、1920×1080 @30fps HEVC、10.27 秒)。被写体は静止中のメリーゴーラウンド(屋外)
- 静止確認: 背景(奥の木々)と回転台(馬)のストリップ別位相相関シフトが 0.2 秒で 34px vs 31px とほぼ一致 — 回転台は稼働していない。写り込む来園者は局所動体(宣言済み汚染)
- 抽出窓: t ∈ [1.0, 7.0) の 6 秒、4fps × 24 frames(元 30fps の index は npz の `src_indices`)。冒頭 1 秒は超クローズアップ、7 秒以降は人物が増えるため除外
- 内容: `fit` = グレースケール 96×54(フィット用)、`fig` = 192×108(図用)、いずれも float32 [0,1]
- 生動画(12 MB)はリポジトリに含めない — 必要なら orange が原本を保持
