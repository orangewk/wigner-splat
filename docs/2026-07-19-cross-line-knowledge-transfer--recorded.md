## 0. 両線が独立に発見した同一現象(核心)

- 量子 exp16(#39): lossy 標的で train NLL 差 ~1e-4 の裏に忠実度差 0.4
  が隠れる(rank-2 崩壊モードに尤度がほぼ盲目)。
- 量子 exp17(#42): fitted-η が NLL 平坦面(3.96110–3.96112)上で
  η 0.56–0.77・F 0.06–0.80 に散る — η はこの設計で識別不能。
- 応用 exp16→20: 18 dB の train loss は欠落構造に盲目(held-out 相関
  +0.03)、27 dB で初めて分散型証明書が残差と相関(+0.33)。

共通構造: **フィット目的関数の平坦方向にこそ品質差が住む**。平坦方向を
形式的に検出する道具は応用線が整備した H(Fisher/GN)解析であり、これが
双方向移転の軸になる。

## 1. 量子 → 応用(応用線のブラッシュアップ候補)

1. **ニュイサンスは「fit するな、測るか固定しろ」**(exp17 の η の教訓)。
   尤度が平坦なニュイサンスの joint fit は ignore より悪化し得る。応用線の
   blur knob ablation 失敗(on ≥ off が 2/3 seed)は同型。GPU ラウンドで
   blur / appearance を再導入するなら、`diagnose_eta.py` 方式の識別可能性
   診断(複数スタート → NLL/loss 平坦性の確認)を先に宣言してから。
2. **選択規則の事前宣言と init-fragility 検査**(exp16 #39)。
   best-by-train-loss の選択規則を宣言し、seed 間の幾何のばらつき自体を
   崩壊モード検出の診断量にする。
3. **容量の飽和スイープ**(exp18 #40 の R=4–5 特定手順)。splat 数 /
   densification 予算を「宣言済み飽和点」で選ぶ手順に移植可能。
4. **アンサンブル分散を Gate B2 の対照群に追加**(最有力・ほぼ無料)。
   3 fit-seed のチェックポイントは既にあるので、seed 間レンダ分散
   (アンサンブル UQ)が計算できる。「H⁻¹ 証明書は力ずくの反復に勝てるか」
   は B2 の本丸の問いであり、round 4 宣言への追加候補として最優先。
5. (PR #66 クロスチェックで指摘済みの再掲)単一スカラー damping への
   σ_pred の感度スイープも round 4 の設計材料。

## 2. 応用 → 量子(ViceVersa)

1. **証明書機構の逆輸入 — 位相空間の信頼度マップ**。σ_pred² = JᵀH⁻¹J の
   機構は画素を (x,p) に読み替えれば Wigner 再構成の誤差バーになる。
   量子線のパラメータ数は小さい(K≤6, R≤5)ので乱択不要・厳密な H が
   組める。#39 の宿題「fidelity-blind な診断量」への直接の答えでもある:
   崩壊モード・η 不識別 = H の平坦方向。特に **W(0,0) 負値の誤差バー**
   (「この負性はノイズではない」の定量化)は物理的に意味のある成果物
   候補。
2. **バイアス/分散の適用条件**(round 2→3 の教訓)。分散型証明書は
   バイアス(モデル不整合)が小さい作業点でのみ機能する。予言:
   purefock3(0.9727、truncation 天井 ~0.993 近傍)や bbdagS には効く
   見込み、splat 再構成(overlap ~0.50、バイアス支配)にはまだ効かない —
   そちらは先にモデル側。
3. **対照群規律(B2 型)**。量子側で不確かさを主張する際も、|W| 振幅・
   ‖J‖ 級の安価な対照との比較を宣言に含める。応用線は「賢い証明書が
   単純対照に勝てない」を 2 回経験済み(Phase 0 の控えめ化、round 3 の
   B2 不合格)。同じ罠の予防。
4. **乱択 matrix-free Fisher(exp20 Phase 4b)**。現行の量子実験には
   不要(厳密 H が組める)が、将来パラメータが増えた場合に備え、
   unbiased Rademacher 推定器 + tiny-exact-gate 検証パターンの実装が
   `20_real_video_gpu/` に検証済みで存在する。

## 3. 実行順の推奨(宣言は各 issue で)

- 応用線 round 4: アンサンブル分散対照 + ノブ識別可能性診断 +
  damping 感度を宣言に含める。
- 量子線 次イシュー候補: 厳密 H による Wigner 誤差バーと W(0,0) 負値の
  有意性判定(適用先は purefock3 / bbdagS から)。
