# wigner-splat v0.1.0 release notes

Status: proposed release notes. Publish these notes with the first GitHub Release.

## Scope

`wigner-splat` is a preliminary, reproducible research prototype for
continuous-variable quantum-state tomography from homodyne measurements. This
release archives the repository's documented experimental record; it does not
claim a universally superior tomography method.

## Included work

- Signed anisotropic Gaussian-mixture reconstruction with a closed-form Radon
  forward model, analytic gradients, and adaptive component birth, splitting,
  and pruning.
- Constructively physical Gaussian-ket models of the form `rho = B B-dagger`,
  with closed-form homodyne likelihoods, analytic gradients, loss modelling,
  and finite-rank extensions.
- Reproducible synthetic multimode benchmarks, iterative `R rho R` MLE
  baselines, tests, committed experiment logs, and figures.
- A public GKP homodyne-data benchmark with source attribution and its original
  README under the applicable data terms.

## Current evidence and limits

- In synthetic two-mode tests, signed splats matched MLE fidelity across 20
  paired seeds while using about 1/7.4 of the measured compute.
- A signed-splat three-mode result is not a physical-tomography result: its
  non-PSD Wigner-overlap score is not state fidelity.
- On public GKP data, constructively physical rank-two squeezed-ket models
  improve on matched-capacity rank-one controls, but remain about
  0.0005-0.0010 nats per held-out sample behind the empirical MLE frontier in
  exploratory reshuffles.

Read [`README.md`](../README.md) for the evidence map and
[`research-log.md`](research-log.md) for protocols, negative results, and limits.
